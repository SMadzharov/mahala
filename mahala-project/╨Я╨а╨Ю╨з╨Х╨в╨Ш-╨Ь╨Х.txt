МАХАЛА — демо проект (за пускане на лаптопа)
================================================

Какво ти трябва:
  Node.js (вече го имаш, ако имаш Claude Code). Провери с команда:  node -v

СТЪПКИ (Windows)
----------------
1. Разархивирай тази папка някъде, напр. на десктопа.

2. Отвори папката във File Explorer.

3. В адресната лента горе (където пише пътя на папката) напиши:
      cmd
   и натисни Enter. Отваря се черен команден прозорец ТОЧНО в тази папка.

4. Напиши и Enter (само първия път, тегли нужното — отнема около минута):
      npm install

5. Напиши и Enter:
      npm run dev

6. Ще се появи адрес, обикновено:
      http://localhost:5173
   Копирай го в браузъра (Chrome/Edge). Сайтът е пред теб.

7. Всяка промяна по кода се вижда веднага в браузъра, без да правиш нищо.

За да спреш сървъра:
   в черния прозорец натисни  Ctrl + C

КЪДЕ Е КОДЪТ
------------
  src/App.jsx   <- целият сайт е тук (всички екрани и функции)
  src/main.jsx  <- стартова точка
  index.html    <- основната HTML страница

ЗА КАЧВАНЕ НА ХОСТИНГ (по-късно)
--------------------------------
  npm run build
  -> създава папка "dist" с готовите файлове, които се качват в public_html.

Забележка: остави интернета включен — шрифтовете и иконките се теглят онлайн.
