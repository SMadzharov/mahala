MAHALA - project files (for GitHub / Netlify)

These files go inside the "mahala-project" folder in your GitHub repo.
Just upload: package.json, package-lock.json, vite.config.js, index.html,
and the whole "src" folder. Do NOT add netlify.toml (not needed).

To run locally:  npm install   then   npm run dev   ->  http://localhost:5173
